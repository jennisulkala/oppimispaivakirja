Tällä viikolla opin miten luoda html-tiedosto (ja perus nettisivun perusta) Visual Studio Coderilla ja miten tuoda se “eloon” GitHubin avulla. Sain selkeyttä siihen, että miten HTML, CSS ja Javascript toimii ja mitkä niiden funktiot on. HTML on siis kaiken sisällön perusta/itse sisältö, CSS antaa sisällölle tyylin ja Javascript tekee sisällöstä interaktiivista. Ymmärsin mihin kohtaa koodia CSS laitetaan , eli style-tagien sisään head-osioon. Tällöin puhutaan sisäisestä CSS:stä (internal CSS), jolloin se vaikuttaa kyseisen HTML-tiedoston tyyliin. Ulkoisesta CSS:stä (external CSS) puhutaan, kun halutaan vaikuttaa useaan HTML-tiedostoon samanaikaisesti. Tällöin ulkoinen CSS on tehty erilliselle tiedostolle, joka taas linkitetään haluttuihin HTML-tiedostoihin, jolloin se vaikuttaa kokonaisen nettisivuston tyyliin.

Tällä viikolla opin myös:
  • h1-h6 tagit on otsikoille
  • p-tagi on leipätekstille
  • img-tag on kuville
  • ul- ja li-tageilla voi tehdä listoja

    • Opin miten koodin sisään voi jättää kommentteja huomioksi itselle tai muille.
